## 1. Proyecto KinalApp
* Aplicación backend desarrollada con Spring Boot orientada a la gestión de un sistema de ventas.
* Permite administrar usuarios, clientes, productos, ventas y detalle de ventas
* mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).
* El sistema maneja relaciones entre entidades, donde cada venta está asociada a un
* cliente y un usuario, e incluye múltiples productos a través del detalle de venta.

## 2. Tecnologías utilizadas
* Java
* Spring Boot
* Spring Data JPA
* MySQL Workbenhc

## 3. Funcionalidades
* Gestión de clientes
* Gestión de usuarios
* Gestión de productos
* Registro de ventas
* Detalle de productos de venta

## 4. Requisitos
* Java JDK 17 o superior
* MySQL Workbenhc
* Editor de codigo

## 5. Estructura del proyecto
* entity → Clases y atributos en la base de datos
* repository → Interfaces JPA
* service → Lógica de negocio
* controller → Endpoints REST
